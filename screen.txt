Pizza.py Screenshots :h3

The screenshots and images on this page are from various Pizza.py
tools.  Click on any small-size image to view a larger version.

"animate"_#animate: image and GUI from animate tool
"gnu"_#gnu: GnuPlot plots from gnu tool
"image"_#image: image thumbnails from image tool
"matlab"_#matlab: MatLab plots from matlab tool
"mdump"_#mdump: mesh file displayed from mdump,gl,vcr tools
"multi"_#multi: multiple tools active simultaneously
"plotview"_#plotview: plots and GUI from plotview tool
"rasmol"_#rasmol: RasMol images from rasmol tool
"raster"_#raster: Raster3d image from raster tool
"svg"_#svg: SVG image from svg tool
"gl"_#gl: GL image from gl tool
"vcr"_#gl: images and GUI from vcr tool :tb(s=:)
